# Pinus Golf!

Nettside for å registrere og følge med på golfrunder på Pinus. Sjekk ut https://pinusgolf.no

### Database

En postgres database i Supabase er satt opp for å lagre spillerinformasjon og golfrunder.

### Bilder

Bilder lagres i en bucket i Supabase.

### CMS

Under utvikling: Sanity brukes for å administrere innholder under 'Høydepunkter'.

### Ikoner

Bruker ikoner fra https://hugeicons.com/icons?style=Stroke&type=Rounded.
